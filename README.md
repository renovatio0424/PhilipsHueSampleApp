# PhillipsHue Android Library

Feature
======
1. connect bridge
2. get list of lights 
3. turn on or off the lights
4. change hue & color of lights

Requirement
======
you should add this properties to application manifests

```xml
//manifests
<application
    android:usesCleartextTraffic="true"/>
```
